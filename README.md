# Color+

Smart color tools for designers.

## Features
- **Contrast Fix** – Fix color contrast issues for accessibility
- **Harmony Finder** – Discover beautiful color schemes
- **Color Modifier** – Fine-tune colors with live adjustments

## Keyboard Shortcuts
- `Ctrl+Enter` – Generate recommendations
- `S` – Swap colors (Contrast Fix)
- `C` – Copy text color
- `B` – Copy background color
- `L` – Lock/unlock text color
- `K` – Lock/unlock background color

## Local Setup
1. Clone the repo
2. Open `index.html` in your browser
3. No build process needed!

## Built With
- HTML5, CSS3, JavaScript
- [Chroma.js](https://gka.github.io/chroma.js/) for color math
